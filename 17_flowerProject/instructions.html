Emma Example

Tasks

  1. Create a folder for the project
  2. Write create statements for the database
  3. Create the database
  4. Create data access layer for the database
  5. Create a console application to fetch, add, update or delete the data

1 Create a folder for the project

Create a folder for the project. Name the folder with Example_Emma_flower_project.
Install mariadb library into the project folder.
Install all necessary libraries to the project.

2 Write create statements for the database

Write create statements to create a database called flowerdb.
Create a table flower, for flower data. Table columns are:
  • flowerId integer
  • name varchar(26)
  • unitPrice integer
  • site varchar(15)
  • stock integer
All columns are mandatory. The primary key is flowerId.
Add two insert statements into the create statements for testing.
Create user leo@localhost with password g0SfVD7s. Grant all necessary privileges to the user.
Save all necessary create statements into a sql file Example_Emma_flower_createStatements.sql

3 Create the database

Create the database with the create statements .

4 Create data access layer for the database

Create a datastorage layer with all necessary methods.

Necessary operations:

Your data access layer should have the following operations:
  • get all
  • get one
  • insert
  • update
  • remove
In the startup phase of the server the data is read from the data file. The data strorage in this version is in memory only, so all changes of the data is done only in memory and they are not saved back to the disk file.

5 Create a console application to fetch, add, update or delete the data

Create a console menu. Menu has all necessary operations.
Choose:
  1. get all flowers
  2. get flower
  3. add flower
  4. update flower
  5. remove flower
  6. exit
Your choise ( 1,2,3,4,5 tai 6):

Choosing wrong number spawns an error message:

###################################
 Choose number 1,2,3,4,5 or 6
###################################

Input library

You can use this function for reading input

  module.exports = prompt => {
    process.stdout.write(prompt);
    return new Promise(resolve => {
      const syote = process.stdin;
      syote.resume();
      syote.once('data', (data) => {
        syote.pause();
        resolve(data.toString().trim());
      });
    });
  };
