2 Write create statements for the database

Write create statements to create a database called gamedb.
Create a table game, for game data. Table columns are:
  • number integer
  • name varchar(53)
  • quantity integer
  • genre varchar(23)
  • rating varchar(9)
All columns are mandatory. The primary key is number.

5 Create REST server (indexREST.js)

Create a REST server. Server is listening host localhost:4000. Allow cross-origin requests (Cross-Origin Resource Sharing).

REST API

Create REST API. The resouces of the REST API are given and returned in json format. The API should include following:

Get all

GET /api/games

Get game

GET /api/games/number

Example:

GET /api/games/1

Delete game

DELETE /api/games/number

Update

PUT /api/games/number

Note: if the resource with the given number doesn't exist, it may be created

Insert game data

POST /api/games
